# Flood-hazard-risk-modelling-using-Random-Forest-Machine-Learning
The project demonstrates an analysis of predicting flood risk zones  and hazard exposure in urban environments using machine learning. We performed a flood risk analysis based on multi-criteria environmental factors using a Random Forest Machine Learning. We extract a set of flood hazard triggers and exposure data from open-source datasets and satellite remote sensing. The selected flood triggers include:
    # Elevation
    # Slope gradient
    # Topographic wetness index (TWI)
    # Flow accumulation
    # Soil type
    # Drainage density
    # Distance from rivers
    # Distance from roads
    # Land cover/use
    # Population density
    # Long-term mean wet season precipitation
